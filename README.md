## 🎯 Current Direction
기존의 웹툰/퀴즈 도우미 수준을 넘어서,
다음 시험/학습 도메인을 통합하는 **AI 기반 반복학습 플랫폼**으로 확장하는 것이 목표입니다.

- AWS SAA
- 전산 7급
- 정보보안기사
- 이후 확장 가능한 자격증/시험 데이터셋

핵심 차별점은 아래와 같습니다.

- **Anki식 반복학습**: 매일 새 문제와 복습 문제를 큐레이션
- **Obsidian식 링크 구조**: 틀린 문제를 개념/키워드/연관 문제 그래프로 확장
- **클라우드·보안 중심 설계**: AWS 기반 운영, 인증/인가, 감사 로그, 관측성
- **AI 학습 추천**: 오답 원인과 취약 개념을 분석해 다음 학습 경로 제안
- **참여형 서비스성**: 랭킹, 시즌제, 챌린지 등으로 학습 몰입도 강화

## 🧱 Current Data Asset
이미 `data/questions.json` 형태의 학습 문제 덤프를 확보해 두었고,
현재 커밋 기준으로는 **객관식 문제 60개 시드 데이터**가 들어 있습니다.

이 점은 포트폴리오에서 매우 중요합니다.
이 프로젝트는 아이디어만 있는 상태가 아니라,
**실제 학습 데이터를 수집·정리하고 서비스 자산으로 전환하기 시작한 프로젝트**로 설명할 수 있습니다.
